Установка

1. Нужно иметь установленные haxe, node.js последних версий.
2. В папке `server` вызывать `npm install`
3. В папке `preview`
	* widnows: вызвать `npm install watch -g`
	* linux, mac: `npm install`

Запуск

1. В папке `server` вызвать `npm start`
2. В папке `preview`
	* widnows: вызвать `start-watch.bat`
	* linux, mac: `npm start`
3. Зайти браузером по адресу: http://localhost:3005/#test/test
